# F1 2024 Season Analyzer

Analyzes the 2024 Formula 1 season using real race data from the FastF1 library.
Computes driver championship standings and visualizes them with matplotlib.

Built as part of a local AI development stack (Python analysis + local LLM summaries).

## Status
Work in progress — standings + chart done; teammate comparison and AI-generated
season summary coming next.

## Scripts
- \season_standings.py\ — pulls the 2024 race schedule
- \driver_standings.py\ — computes final driver points, saves to CSV
- \points_chart.py\ — charts the standings, saves a PNG
